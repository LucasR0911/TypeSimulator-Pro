# 项目介绍（TypeSimulator V9）

TypeSimulator V9 是一个面向“输入回放演示”的 Windows 工具，核心目标不是单纯把字打出来，而是把过程做得更像真人。

它支持按字符回放、错字修正、节奏变化、疲劳变化和标点停顿，适合用于录屏、直播演示、脚本复现、产品讲解等需要“可控输入过程”的场景。

## 这个版本解决了什么问题

很多同类工具的回放节奏太机械，尤其是逗号句号后没有自然停顿，看起来一眼假。V9 新增了“标点发呆停顿”：

- 可单独开关
- 可调停顿上限（100ms~3000ms）
- 在 `.` `,` `。` `，` 后随机增加停顿

这样在句子断点会出现轻微“想一下再继续打”的感觉，观感明显更自然。

## 功能组成

- 基础回放：按设定 CPM 输出文本
- 随机延迟：降低节奏规律性
- 错字模拟：打错、退格、重打
- 双击修正：偶发重复输入后撤销
- 错字后犹豫停顿：0.5~1 秒 + 可调触发概率
- 节奏曲线：开头慢、中段快、结尾慢
- 疲劳曲线：长文本后半段节奏下滑
- 标点发呆停顿：V9 新增
- 进度与剩余时长显示
- 配置自动保存

## 使用建议

- 演示型场景：速度 120~180 CPM，随机延迟开启
- 强拟人：错字 3~8%，标点停顿上限 800~1500ms
- 长文回放：建议开启节奏曲线和疲劳曲线

## 技术说明

- 平台：Windows
- 框架：.NET Framework 4.8（WPF）
- 入口工程：`TypeSimulator/TypeSimulator.sln`
